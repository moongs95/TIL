# Troble shoot

## Working Directory

```shell
git stash
git stash pop
git stash list
git stash drop {index}
```

```shell
mv a.md to b.md
```

```shell
git restore {filename}
```

## Staging Area

```shell
git restore --staged {filename}
```

## Local Repository

```shell
git commit --amend
```

```shell
git revert --no-commit HEAD~{nums of commit}..
git commit
git push origin <branch>
```
